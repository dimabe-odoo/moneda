Odoo Chile :: Actualiza Tipos de Cambio desde API SBIF

[l10n_cl_currency_rate_update]

Automatiza la actualización de tipos de cambio para USD, EUR, UF y UTM

Funciones

☑ Crea las monedas UF, UTM y ajusta atributos de CLP
☑ Requiere de "Currency Rate Inverted" para CLP
☑ "API Key" se actualiza en Configuración general de Contabilidad

Autores

Intellego-BI.com

Resumen

Automatiza el proceso de actualización diaria de tipos de cambio para USD, EUR, UF y UTM empleando el servicio web de la Superintendencia de Bancos e Instituciones Financieras (SBIF) de Chile. Se debe solicitar un "API Key" a la SBIF ingresando a este enlace (SBIF API). Una vez obtenido el "API Key" se debe actualizar en la Configuración General del menú de Contabilidad.

Dependencias

  • Odoo - Base [base]
  • Odoo - Correo [mail]
  • Odoo - Facturación [account]
  • OCA - Currency Rate Inverted [currency_rate_inverted]
